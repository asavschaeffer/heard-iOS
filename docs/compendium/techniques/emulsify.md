## Overview
Combine immiscible liquids into a stable mixture using shear and emulsifiers.

## Rules
Add fat slowly while whisking or blending to build stability.

## When to Break
Create a temporary emulsion when a short-lived sauce is acceptable.

## Notes
Common in vinaigrettes, mayo, and pan sauces.
